# Contact App

## Installation

**Required [Flutter](https://flutter.dev/docs/get-started/install) library**

OS X, Windows & Linux:

```
flutter packages get
```

## Run

```
flutter run
```

## Description

_The objective of this project is create contact book._

## Development setup

This project was developed with Flutter and Dart.
